# ข้อจำกัดความรับผิดชอบ

โปรเจกต์นี้เผยแพร่เป็นโอเพนซอร์สภายใต้ [MIT License](https://opensource.org/licenses/MIT) ซึ่งเปิดให้ทุกคนสามารถนำไปใช้งานหรือดัดแปลงได้อย่างอิสระ แต่มีข้อควรทราบดังนี้:

**ให้บริการแบบ "ตามสภาพ" (As-is):** ระบบนี้ไม่มีการรับประกันใดๆ ทั้งสิ้น ไม่รับประกันว่าจะปราศจากบั๊ก หรือรับประกันว่าตัว Widget จะสามารถเข้าถึงได้ตลอดเวลา

**การอัปเดตและ Breaking Changes:** ปัจจุบัน Custom Widget นี้ถูกโฮสต์ไว้บน [Cloudflare Pages](https://pages.cloudflare.com/) แม้ระบบนี้จะค่อนข้างเสถียร แต่ในอนาคตอาจมีการอัปเดตเพื่อปรับปรุงหรือเพิ่มฟีเจอร์ใหม่ๆ ซึ่งอาจทำให้เกิด Breaking Changes ที่ส่งผลกระทบต่อรูปแบบการใช้งานเดิมของคุณได้ หากคุณนำไปใช้งานจริงจังและกังวลเรื่องผลกระทบจากการอัปเดต ขอแนะนำอย่างยิ่งให้ทำการโคลนโค้ดไปโฮสต์และจัดการ URL ด้วยตัวคุณเอง (ดู {doc}`self-hosting`) เพื่อให้คุณสามารถควบคุมระบบทั้งหมดอย่างสมบูรณ์

**การสนับสนุน (Support):** หากคุณพบปัญหาในการใช้งาน สามารถเข้าไปเปิด [Issue](https://github.com/dtinth/bizdocgen/issues) แจ้งไว้ได้ที่ GitHub Issues อย่างไรก็ตาม ผู้พัฒนาอาจไม่ได้มีทรัพยากรพอที่จะดูแลหรือแก้ปัญหาให้ได้ทั้งหมด แต่เนื่องจากโปรเจกต์นี้เป็นโอเพนซอร์ส คุณสามารถ Fork โปรเจกต์ไปแก้ไขได้เลย
