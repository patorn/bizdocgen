# bizdocgen

bizdocgen เป็น Custom Widget สำหรับ [Grist](https://www.getgrist.com/) ที่ช่วยให้คุณสร้างและพิมพ์เอกสารธุรกิจ เช่น ใบเสนอราคา ใบแจ้งหนี้ และใบเสร็จรับเงิน ได้โดยตรงจากข้อมูลใน Grist

สำหรับที่มาและเรื่องราวเบื้องหลังโปรเจกต์นี้ อ่านได้ที่ [dt.in.th/Bizdocgen](https://dt.in.th/Bizdocgen)

## ความสามารถหลัก

- **สร้างเอกสาร 3 ประเภท** คือ ใบเสนอราคา, ใบแจ้งหนี้, ใบเสร็จรับเงิน
- **คำนวณภาษีอัตโนมัติ** โดยเวลารับเงินจากบริษัทสามารถระบุให้หักภาษี ณ ที่จ่าย ได้เลย
- **สร้าง QR Code พร้อมเพย์ พร้อมฝังจำนวนเงิน** เพื่อความสะดวกในการรับชำระเงิน
- **พิมพ์เอกสารขนาด A4** จัดหน้าพร้อมพิมพ์ทันที
- **เปลี่ยนฟอนต์ สี และรูปแบบตามต้องการ** โดยการปรับแต่ง CSS
- **รองรับ Markdown** ในรายละเอียดสินค้าและหมายเหตุ
